# 快速开始指南

## 第一次使用？跟着这 5 步走！

### 步骤 1：安装 Python 依赖 📦

打开命令提示符（CMD），进入项目目录，运行：

```bash
pip install -r requirements.txt
```

等待安装完成（可能需要几分钟）。

---

### 步骤 2：获取豆包 API Key 🔑

1. 访问 [豆包 AI 控制台](https://console.volcengine.com/ark)
2. 注册/登录账号
3. 点击"创建 API Key"
4. 复制生成的 API Key（格式类似：`xxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx`）

---

### 步骤 3：配置环境变量 ⚙️

1. 复制 `.env.example` 为 `.env`：
   ```bash
   copy .env.example .env
   ```

2. 用记事本打开 `.env` 文件

3. 找到这一行：
   ```ini
   DOUBAO_API_KEY=your_doubao_api_key_here
   ```

4. 把 `your_doubao_api_key_here` 替换成你的 API Key：
   ```ini
   DOUBAO_API_KEY=xxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
   ```

5. 保存文件

---

### 步骤 4：启动应用 🚀

双击 `run.bat` 文件，或在命令行运行：

```bash
python run.py
```

看到以下信息说明启动成功：

```
✅ 依赖包检查通过
✅ 环境变量配置检查通过
✅ 目录已创建
🚀 启动应用...
应用已启动: http://127.0.0.1:8000
```

---

### 步骤 5：开始使用 🎉

1. 打开浏览器，访问：http://127.0.0.1:8000

2. 点击 **"手动获取最新论文"** 按钮
   - 等待 1-2 分钟，应用会获取并初筛最新论文

3. 浏览论文列表
   - 每篇论文都有 AI 生成的 2 句话简介
   - 可以查看标题、作者、点赞数

4. 选择感兴趣的论文
   - 勾选最多 6 篇论文

5. 点击 **"生成精读报告"** 按钮
   - AI 会生成深度解读
   - 浏览器会自动打开 6 个标签页，每个显示一篇论文的精读

6. 阅读精读报告
   - 包含研究背景、核心问题、方法、实验结果等
   - 可以点击链接查看原文

## 常用操作

### 设置开机自启动

如果希望电脑开机时自动启动应用：

1. 右键 `install_startup.bat`
2. 选择 **"以管理员身份运行"**
3. 按提示完成安装

### 修改定时时间

编辑 `.env` 文件，修改这一行：

```ini
SCHEDULE_TIME=11:00    # 改成你想要的时间，如 14:30
```

### 查看日志

如果遇到问题，可以查看 `logs/` 目录下的日志文件：

- `app_YYYY-MM-DD.log`：应用主日志
- `scheduler_YYYY-MM-DD.log`：定时任务日志

## 遇到问题？

### 问题 1：提示缺少依赖包

**解决**：
```bash
pip install -r requirements.txt
```

### 问题 2：API 调用失败

**解决**：
- 检查 `.env` 文件中的 `DOUBAO_API_KEY` 是否正确
- 确认 API Key 有足够的额度

### 问题 3：论文列表为空

**解决**：
- 点击"手动获取最新论文"按钮
- 等待 1-2 分钟
- 如果还是空的，查看 `logs/` 目录下的日志

### 问题 4：周末没有论文

**说明**：HuggingFace 通常在工作日更新，周末可能没有新论文
